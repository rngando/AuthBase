
# 🧩 Plano Específico

## Objetivo
Criar um CRUD/API simples com as seguintes funcionalidades:

- Autenticação JWT
- Permissões (admin / user)
- Validação de dados
- Paginação e Filtro
- Logs de erros

## 💣 Diferenciais
- Documentação com Swagger
- Testes básicos

## 👉 Resultado
Transformar em um backend "profissional", não apenas um exercício.
